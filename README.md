


## Installation
1. Clone this repository or download zip
```bash
git clone https://github.com/itzibtihal/Wiki.git
```
2. run the following CMDs in vsCode terminal 
```bash
    composer init 
```
3. 

 ```bash
    composer require vlucas/phpdotenv 
``` 
4. 

 ```bash
    composer dump-autoload 
```
